Language Used :  PHP 5.62 (Developed in Core PHP)
Database :  My SQL
User Interface Design :  HTML, JAVASCRIPT,CSS
Web Browser:  Mozilla, Google Chrome, IE8,OPERA
Software :    XAMPP Server
